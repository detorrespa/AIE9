# Datos para el RAG

Coloca aquí archivos **PDF** que quieras indexar para el Retrieval-Augmented Generation.

El RAG cargará todos los PDFs de esta carpeta (y subcarpetas) al iniciar.

**Ejemplos de documentos útiles:**
- PDFs sobre salud felina (vacunas, nutrición, etc.) — el agente está preparado para estas preguntas
- Cualquier PDF técnico o de conocimiento que quieras consultar
- Papers de arXiv exportados en PDF

**Para probar:** Descarga un PDF y guárdalo en esta carpeta, luego ejecuta `python main.py`.
